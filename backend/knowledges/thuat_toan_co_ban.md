# Thuật toán cơ bản

Thuật toán là tập hợp các bước để giải quyết một bài toán.

---

# Linear Search

Linear search tìm kiếm từng phần tử.

Pseudo code:

for each element
if element == target
return index

Độ phức tạp:

O(n)

---

# Binary Search

Binary search hoạt động trên mảng đã sắp xếp.

Các bước:

1 chia mảng làm hai
2 so sánh phần tử giữa
3 loại bỏ một nửa

Độ phức tạp:

O(log n)

---

# Bubble Sort

Bubble sort sắp xếp bằng cách hoán đổi.

for i
for j
if arr[j] > arr[j+1]
swap

Độ phức tạp:

O(n^2)
