# Kiến thức lập trình hướng đối tượng

Lập trình hướng đối tượng (Object Oriented Programming) là phương pháp tổ chức chương trình dựa trên đối tượng.

---

# Class

Class là khuôn mẫu để tạo ra đối tượng.

Ví dụ:

class Student
{
   int id;
   string name;
};

---

# Object

Object là thể hiện của class.

Ví dụ:

Student s1;

---

# Encapsulation

Encapsulation là nguyên lý đóng gói dữ liệu.

Dữ liệu được bảo vệ bằng cách sử dụng private.

---

# Inheritance

Inheritance cho phép lớp con kế thừa lớp cha.

Ví dụ:

class Animal
class Dog : public Animal

---

# Polymorphism

Polymorphism cho phép một hàm có nhiều dạng khác nhau.

Ví dụ:

Method Overloading  
Method Overriding

---

# Abstraction

Abstraction giúp ẩn chi tiết cài đặt.

Người dùng chỉ cần sử dụng interface.
